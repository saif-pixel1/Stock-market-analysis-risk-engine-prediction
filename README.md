# 📈 Advanced Quant Trading System v2.0

A powerful multi-market stock analysis dashboard built using Streamlit, Plotly, and Yahoo Finance API.

This system provides:
- 📊 Risk Metrics Analysis
- 📉 Volatility & Drawdown Insights
- 🎲 Monte Carlo Simulations
- 🌍 Indian + Global Market Coverage
- ⚡ Interactive & Modern UI

---

## 🚀 Features

### 🏠 Dashboard
- Portfolio-wide performance overview
- Top 10 performing stocks
- Highest volatility stocks
- Sharpe Ratio insights

### 📊 Risk Metrics
- Annual Returns
- Annual Volatility
- Sharpe Ratio
- Maximum Drawdown
- Custom stock ranking filters

### 🎲 Monte Carlo Simulation
- Select any stock
- Choose forecast days
- Run up to 5000 simulations
- View:
  - Expected Price
  - 5% Worst Case
  - 95% Best Case

### 🌍 Markets Covered

#### 🇮🇳 Indian Stocks (NSE)
Includes:
RELIANCE, HDFCBANK, TCS, INFY, SBIN, BAJFINANCE, LICI, NTPC, etc.

#### 🌎 Global Stocks
Includes:
NVDA, AAPL, MSFT, TSLA, AMZN, META, JPM, XOM, DIS, etc.
